## Download and Installation

```sh
git clone https://github.com/gabrielaimeeg/word_counter.git word_counter
```

```sh
cd word_counter
```

```sh
java -jar target/dependency/webapp-runner.jar target/*.war
```
It will be running in you localhost:8080
